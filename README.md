Project Title:
Forecasting of Smart City Traffic Patterns using Data Science and Machine Learning

Description:
This project predicts traffic flow across smart city junctions using Machine Learning techniques. Historical traffic data was analyzed to identify traffic trends, peak hours, weekday-weekend variations, and holiday impacts. A Random Forest Regressor was used to forecast future traffic patterns and support intelligent traffic management and infrastructure planning.

Technologies:
Python
Pandas
NumPy
Matplotlib
Scikit-Learn
Streamlit
Holidays

Results:
MAE: 2.39
RMSE: 3.54
R² Score: 0.969
